三、XML约束

1.意义：编写一个文档约束XML写法，防止乱写。
2.技术：XML DTD 与 XML Schema（较难）
3.做框架写dtd
一、DTD（document type definition）

<?xml version ="1.0" encoding="UTF-8" ?>
<!DOCTYPE bookshelf SYSTEM "Book.dtd">
<bookshelf>
     <book >
        <name> 十万个为什么</name >
        <author> 科学家</author >
        <price> 1008</price >
     </book >
     <book >
        <name> 蓝猫</name >
        <author> 葛平</author >
        <price> 12</ price>
     </book >
</bookshelf>

Book.dtd文件
<!ELEMENT bookshelf (book+ )>
<!ELEMENT book (name, author,price )>
<!ELEMENT name (#PCDATA)>
<!ELEMENT author (#PCDATA)>
<!ELEMENT price (#PCDATA)>

注意ELEMENT的大写，#PCDATA是字符串意思，也大写。book里面的属性必须全部匹配上。
